📚 Smart Study Planner

A Smart Study Planner built with Python and Streamlit.
Add your tasks, set deadlines and difficulty level — 
AI will automatically create a personalized study 
timetable for you!

🚀 Features:
- Add tasks with deadline and difficulty rating
- Auto calculates task priority
- Labels tasks as Critical, Important or Comfortable
- AI generates smart study timetable
- Progress tracker
- Motivational messages from AI

🛠️ Built With:
- Python
- Streamlit
- SQLite
- Groq AI API

💡 Built during a 24-hour Hackathon!
